# Etch-a-Sketch

## Overview

This project is a browser-based Etch-a-Sketch application built using HTML, CSS, and JavaScript. The goal of this project was to strengthen my understanding of JavaScript fundamentals, DOM manipulation, event handling, and dynamic element creation.

The application allows users to draw on a customizable grid by clicking and dragging across the sketchpad.

## How It Works

* A grid of square divs is dynamically generated using JavaScript.
* Users can draw by clicking and dragging the mouse across the grid.
* Each square changes to a random RGB color when interacted with.
* Squares progressively darken by 10% with each interaction until fully opaque after 10 passes.
* Users can generate a new grid size using the "New Grid" button.
* Grid sizes can be customized up to 100x100 squares.
* The "Clear" button resets the sketchpad without changing the grid size.

## Features

* Dynamically generated grid system
* Adjustable grid sizes up to 100x100
* Flexbox layout implementation (without CSS Grid)
* Click-and-drag drawing functionality
* Random RGB color generation
* Progressive darkening effect
* Clear/reset functionality
* Interactive browser-based UI

## How to Run

* Clone or download the repository
* Open the project folder
* Open `index.html` in your browser
* Click and drag on the grid to draw
* Use the "New Grid" button to resize the sketchpad
* Use the "Clear" button to reset the drawing surface
